Simple git-guide

simple guide to use only some required commands for the basic push (dummies)

  • clone a repo being made or used

  • cd into dir

Set up your user name + email.

    > git config --global user.name “you”
    > git config --global user.email “your@anyserver.xyz”

add the new files(or edited) to the cloned dir

    > git add -A

check status

    > git status

commit

    > git commit -m “your comment”
    > git push

enter your github credentials
